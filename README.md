# 🔴 LiveDeck PRO Aggregator

**LiveDeck** is a high-performance, real-time multi-platform chat aggregator designed for professional streamers. It combines direct WebSocket connections for Twitch and Kick with an integrated "Pro" mode for Restream/Streamlabs, giving you a single, unified, and honest view of your entire community.

## ✨ Features

- **⚡ Real-Time Only**: No simulated data, no fake bots. 100% authentic community engagement with zero delay.
- **🎮 Premium Console UI**: Designed with a sleek, minimalist aesthetic. Features tactile animations, dynamic brand colors, and a flat, distraction-free interface.
- **🛡️ Pro Aggregator Mode**: Seamlessly integrates external URLs (like Restream or Streamlabs) via a secure Iframe. Includes an innovative "Freeze Shield" to safely pause fast-moving external chats.
- **📡 Direct Connections**: Native WebSocket integration for Twitch and Pusher integration for Kick.
- **🧠 Smart Memory**: Automatically saves your channel names and URLs using local storage so your setup is instantly ready for your next stream.

## 🚀 How to Use

1. **Select Platforms**: Choose which platforms you are streaming to (Twitch, Kick, YouTube, Instagram, TikTok).
2. **Enter Channels**: Provide your exact usernames for Twitch and Kick.
3. **Pro Link (Optional)**: If you are using YouTube, TikTok, or Instagram, paste your Restream or Streamlabs chat widget URL into this field.
4. **Go Live**: Click the "GO LIVE" button to launch your dashboard. 
5. **Control Your Chat**: Use the Pause and Clear buttons in the top right to easily manage the flow of messages while you are live.

## 🛠️ Technology Stack
- **Frontend**: Pure Vanilla HTML5, CSS3, and JavaScript (Zero Bloat).
- **Real-Time Data**: Twitch IRC WebSocket, Kick Pusher API.
- **Design System**: Custom CSS variables, responsive layouts, and fluid typography.
